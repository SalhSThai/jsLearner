- จงสร้างตัวแปร summary โดยใช้ค่าจากตัวแปร sales
- netPrice = price x discount (ถ้าไม่มี discount ให้ถือว่า netPrice = price)


```js
const sales = [
  { netPrice: 1000 ,discount 0.1}, 
  { netPrice: 500 ,discount 0.05},
  { netPrice: 100 }
];
const summary = [
  { netPrice: 900 }, 
  { netPrice: 475 },
  { netPrice: 100 }
];
```
